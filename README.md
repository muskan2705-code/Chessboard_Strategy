# Chessboard Strategy Website

Static marketing website for Chessboard Strategy.

## Project Structure

- `index.html`: Homepage
- `about.html`: About page
- `services.html`: Services page
- `insights.html`: Blog listing page
- `blog-1.html` to `blog-6.html`: Individual blog pages
- `contact.html`: Contact page
- `testimonials.html`: Testimonials page
- `team.html`: Team page
- `case-studies.html`: Case studies page
- `vision-mission.html`: Vision and mission page
- `privacy-policy.html`: Privacy policy page
- `styles.css`: Main stylesheet
- `assets/images/`: Site images, logos, and favicons
- `assets/videos/works/`: Homepage work showcase videos
- `*.js`: Frontend interaction scripts and animations

## Branding Assets

- Main site logo currently uses the white Chessboard Strategy logo from `assets/images/`
- Favicons use:
  - `assets/images/favicon-black.png`
  - `assets/images/favicon-white.png`

## How To Edit

1. Open the project folder.
2. Edit the HTML page you want to update.
3. Update shared styling in `styles.css` if needed.
4. Place any new website images inside `assets/images/`.
5. Place any new website videos inside `assets/videos/works/`.

## Notes

- This is a static site, so no build step is required.
- Keep image and video references inside the local `assets/` folder.
- If you preview locally, open `index.html` in a browser or use a simple static server.
